# Brand & Professional Identity: JAB Digital Studio

## Visual Identity (Brand Book) & Color Tokens
- **Palette Source:** Based on provided palette image (Neutrals to Deep Blue).
- **Strategy:** Always use Semantic Color Tokens. Never use hardcoded color names.

### Color-to-Token Mapping
- **Neutral Background (`--color-surface-base`):** The very light grey container from the image.
  - *Usage:* Page backgrounds, section containers.
- **Surface Accent (`--color-surface-card`):** The very light, almost white-grey first swatch.
  - *Usage:* Card backgrounds in Bento Grids.
- **Secondary / Informational (`--color-secondary`):** The light sky blue swatch.
  - *Usage:* Secondary buttons, subtle highlights.
- **Primary / Action (`--color-primary`):** The deep, vibrant blue swatch.
  - *Usage:* Primary CTAs, active states, key text elements.
- **Text / Darkest Accent (`--color-text-base`):** The solid black last swatch.
  - *Usage:* Main body text, darkest borders.

### Design Philosophy
- **Tokens over Colors:** Use semantic names (primary, surface-card) to allow future re-branding.
- **Palette Direction:** Professional, trust-building, corporate-executivel.
- **Layouts:** "Bento Grid" (asymmetric grids).
- **Visuals:** High backdrop-blur (Glassmorphism) with thin borders (`border-text-base/10`).
